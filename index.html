Enersport

Client wanted to create a new website for their company from scratch. I used a modular approach to speed up the Grav CMS, to expand the functional microblog I used collections and taxonomies with TWIG.

    This project was divided into several stages:
  • - draw a new design for the site.
  • - improve old logo, draw new markers and other UI elements;
  • - convert created design to HTML
  • - power it up by Grav CMS
  • - extend the microblog functionality using php template engine TWIG scripts
  • - use the map as a background via google maps API, and marker that I drew for enersport
  • - retina ready images
  • - some javascript/jquery for sliders, form validator, preloader, calculators, clients logos watermark, etc
  • - multilanguage support.

Home page consists of a full-screen slider, the benefits of the company using auto-scroll and CSS3 animations, customers logos protected from downloading using jQuery, google map used as a background for newsletter and the related php script.

About page consists of main information, newsletter, benefits and footer.

Blog page includes microblog with featured post. Grav has no such plugin so I wrote this functionality manually using php. Script selects a random post in the category "featured" and displays it with different CSS style.

Contact page consists of a map with the drawn marker, which is displayed as a background and a form which includes the validator, php script and "thank you" note.
